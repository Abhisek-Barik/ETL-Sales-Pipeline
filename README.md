# ETL Sales Data Pipeline

## Overview
This project demonstrates an end-to-end ETL pipeline using Python and PostgreSQL.

## Tech Stack
- Python (pandas)
- PostgreSQL
- SQLAlchemy

## Steps
1. Extract data from CSV
2. Transform data (cleaning, handling nulls, feature creation)
3. Load data into PostgreSQL
4. Perform SQL analysis

## Key Features
- Data cleaning and preprocessing
- Automated data loading
- SQL-based analysis

## Sample Queries
- Total revenue calculation
- Product-wise sales aggregation

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run pipeline:
   python main.py
